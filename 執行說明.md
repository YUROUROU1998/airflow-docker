# Environment
Docker Desktop, 
macOS


# Run on the Terminal

## 1. Initialize airflow

```bash
pwd #find path
cd /path/airflow-docker
docker compose up airflow-init
```

Wait for the following message before proceeding: 
```
airflow-init-1 exited with code 0
```

---

## 2. Start airflow

```bash
docker compose up -d
```

## 3. Add connection

1. Open to http://localhost:8080
2. Login: `airflow` / `airflow`
3. Go to Admin → Connections
4. Add a new connection:
   - **Connection Id**: `fs_default`
   - **Connection Type**: `File (path)`
5. **Save**

## 4. Create dir, copy sample data to docker
```bash
docker exec airflow-docker-airflow-worker-1 mkdir -p /tmp/test /tmp/success
docker cp sample_data/record_20260423.txt airflow-docker-airflow-worker-1:/tmp/test/record_20260422.txt
docker cp sample_data/record_20260424.txt airflow-docker-airflow-worker-1:/tmp/test/record_20260423.txt
```
